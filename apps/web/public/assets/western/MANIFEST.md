## Usage

### BabylonJS

```typescript
import { SceneLoader } from "@babylonjs/core/Loading/sceneLoader";
import "@babylonjs/loaders/glTF";

// Load a single asset
const result = await SceneLoader.ImportMeshAsync(
  "",
  "/inbox/western-assets/furniture/",
  "piano.glb",
  scene
);
const piano = result.meshes[0];

// Materials are automatically applied from embedded textures
// You can modify materials after loading:
piano.getChildMeshes().forEach(mesh => {
  if (mesh.material) {
    // Adjust material properties
    mesh.material.backFaceCulling = true;
  }
});
```

### Three.js

```javascript
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader';

const loader = new GLTFLoader();
loader.load('/inbox/western-assets/furniture/piano.glb', (gltf) => {
  scene.add(gltf.scene);

  // Textures and materials are automatically loaded
  // Modify if needed:
  gltf.scene.traverse((child) => {
    if (child.isMesh) {
      child.castShadow = true;
      child.receiveShadow = true;
    }
  });
});
```

### Gunslinger + Revolver Attachment

```typescript
import { SceneLoader } from "@babylonjs/core";

// Load both assets (both are now properly scaled)
const [gunslingerResult, revolverResult] = await Promise.all([
  SceneLoader.ImportMeshAsync("", "/inbox/western-assets/characters/", "gunslinger.glb", scene),
  SceneLoader.ImportMeshAsync("", "/inbox/western-assets/weapons/", "revolver-fixed.glb", scene)
]);

const gunslinger = gunslingerResult.meshes[0];
const revolver = revolverResult.meshes[0];

// Find the right hand bone (Mixamo naming)
const skeleton = gunslingerResult.skeletons[0];
const rightHandBone = skeleton.bones.find(b => b.name === "mixamorig:RightHand");

// Attach revolver to hand - transforms are pre-baked!
revolver.attachToBone(rightHandBone, gunslinger);

// Both assets are properly scaled:
//   - Gunslinger: 1.76m tall
//   - Revolver: 0.30m long, grip at origin
//   - Revolver pre-rotated for correct hand orientation
// No manual scale/rotation adjustment needed!
```

### Material Customization

All assets include:
- **UV coordinates** for texture mapping
- **Normals** for lighting calculations
- **PBR materials** with base color textures

To recolor or customize materials in BabylonJS:

```typescript
// Clone material for unique instance
const newMat = mesh.material.clone("customMaterial");
mesh.material = newMat;

// Tint the texture
newMat.albedoColor = new BABYLON.Color3(0.8, 0.6, 0.4);

// Or replace texture entirely
newMat.albedoTexture = new BABYLON.Texture("/custom/texture.png", scene);
```

## Technical Specifications

### Export Settings
- **Format**: glTF 2.0 Binary (.glb)
- **Textures**: Embedded (no external dependencies)
- **Coordinate System**: Y-up (glTF standard)
- **Units**: Meters

### Gunslinger Rig (Mixamo Standard)
```
mixamorig:Hips
├── mixamorig:Spine
│   ├── mixamorig:Spine1
│   │   └── mixamorig:Spine2
│   │       ├── mixamorig:Neck → mixamorig:Head
│   │       ├── mixamorig:LeftShoulder → mixamorig:LeftArm → mixamorig:LeftForeArm → mixamorig:LeftHand
│   │       └── mixamorig:RightShoulder → mixamorig:RightArm → mixamorig:RightForeArm → mixamorig:RightHand
├── mixamorig:LeftUpLeg → mixamorig:LeftLeg → mixamorig:LeftFoot
└── mixamorig:RightUpLeg → mixamorig:RightLeg → mixamorig:RightFoot
```

**Hand bones for attachment:**
- `mixamorig:RightHand` - Primary weapon hand
- `mixamorig:LeftHand` - Support hand

### Revolver Coordinate System
```
revolver-fixed.glb orientation:
       +Z (barrel tip)
        ↑
        |
   -----+----→ +X
        |
        ↓
       -Z (grip at origin)
```
